使用指南
了解如何使用 PSD 智能对象服务

操作步骤

  1. 先配置 API 地址与超时，点击健康检查。
  2. 如需 Photoshop 支持，先"启动 Photoshop"再检测状态。
  3. 分析 PSD 结构（/analyzePsd），确认智能对象信息。
  4. 填写处理参数，执行 /processPsd 完成替换导出。
  5. 调试输出区查看最近请求和错误信息。

注意事项

  • 路径需为本机有效 Windows 路径（示例：D:\path\file.psd）
  • 避免包含 <>:"|?* 等非法字符
  • 确保 Photoshop 已正确安装并可访问
  • 建议以管理员权限运行服务，避免权限问题
